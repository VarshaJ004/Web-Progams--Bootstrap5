

## Web Programming Using Bootstrap 5

### Objective

This project contains responsive web pages developed using **HTML5** and **Bootstrap 5**. The applications are designed with a clean and user-friendly interface while following responsive web design principles.

### Technologies Used

* HTML5
* Bootstrap 5
* CSS

### Bootstrap Components Used

* Container
* Grid System (Row and Column)
* Forms
* Cards
* Buttons
* Images
* Form Controls
* Navigation Links (`<a>` tag)
* Responsive Layout
* Text Alignment Utilities
* Margin and Padding Utility Classes

### Features

* Responsive design for mobile, tablet, and desktop devices.
* Bootstrap Grid System for proper alignment.
* Professional form layouts.
* Card-based user interface.
* Navigation between pages using hyperlinks.
* Clean spacing and styling using Bootstrap utility classes.

### Files Included

* HTML files
* Bootstrap CSS (CDN)
* Images (local/online)
* README.md

### Bootstrap Concepts Practiced

* Responsive Containers
* Rows and Columns
* Form Design
* Card Components
* Buttons
* Image Styling
* Utility Classes
* Hyperlinks Between Pages
* Responsive Design

### How to Run

1. Download or copy all project files into the same folder.
2. Open the HTML file in any web browser.
3. Navigate between pages using the provided buttons or links.
